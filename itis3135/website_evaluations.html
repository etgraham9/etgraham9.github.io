Website Evaluations

Here we will evaluate different webpages based on the CRAP principles of web design.

Arngren

arngren.net is a Norweigen website first designed in 2002 by Arngren Electronics. The purpose of this website was to sell various odd electronic gadgets.

  • Contrast: When it comes to color contrast arngren does a relatively good job in ensuring that text is visible and problems only occur when text is placed over images. arngren's size contrast while vast does not work in a cohesive way to make it effective.
  • Repetition: There is virtually no repetition in this website other than its use of default colors for text without rhyme or reason. The only consistent thing on this website is it's adherance to chaos.
  • Alignment: All the text is left justified but every box is aligned in whatever way it wants creating a collage effect across the whole website.
  • Proximity: Everything in this website is close together. If there are catagories it is not aparent from looking at it.

Youtube

youtube.com is a massively popular video sharing site owned by google. While it has been around for a long time it has received numerous UI updates keeping it up to date with current design principles.

  • Contrast: The majority of the website is black and white with the exceptions being logos, user profiles, and videos. This allows the main content of the site, the videos, to standout as the most improtant feature.
  • Repetition: Youtube repeats a consistent format for user videos using a specific layout to display thier information.
  • Alignment: The site has a navigation bar that is aligned nicely with subscribed channels and all the videos are aligned in a grid making looking at teh videos easy.
  • Proximity: Youtube a has a dense but nicely spaced layout. There are a lot of buttons and information being displayed on the screen but it is arranged in an easy to understand way.
